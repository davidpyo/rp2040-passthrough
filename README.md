# rp2040-passthrough


This is a RP2040 passthrough library intended for use with am32.ca and the Windows configuration app for am32. 

To use with platformio, simply include 

lib_deps = https://github.com/davidpyo/rp2040-passthrough.git in the platformio.ini

For a minimal usage of this library see https://github.com/davidpyo/minimal_rp2040_passthrough

This code is taken from the https://github.com/Project-Stinger/Firmware/tree/main firmware, which was written by Bastian. Thank you for sharing your work! I have just added some bug fixes.
